Movie Recommendation System

Business Objective:Provides personalised movie recommendations for customers based on clusters created by identifying the underlying characteristics of each customer.

Tools Used: Python, Streamlit, Pycharm, Machine Learning Algorithms (Clustering)

Approach: Using the Movielens dataset, multiple datasets were merged, redundant columns dropped, and null values were treated. Feature engineering was done to create additional relevant features to improve recommendations. Using cosine similarity in the Nearest Neighbors algorithm five most similar movies were found. Finally, UI was built using streamlit and the model was deployed.




Cost Prediction Modelling

for Customer Acquisition

Business Objective: Predicted the cost spent on different media marketing platforms for acquiring a customer and evaluated the efficiency of current marketing strategies.

Tools Used: Python, Machine Learning Algorithms(Regression)

Approach: Processed, cleansed, and handled outliers and missing values in the data. Performed feature engineering and statistical testing using the inferences from the univariate and bivariate analysis. Then parametric tuning using GridSearchCV was done for the ensemble models like Adaboost, Gradient Boost, and XGboost following the building of numerous base models (including linear regression, decision tree regressor, random forest, etc). Selected the decision tree regressor model as the final model because of its high R-squared and RMSE values.




Sales Insights Dashboard

Dynamic analysis of Amazon Sales across nine clothing categories in India during Jan 2022 - Dec 2022 to uncover valuable insights and optimize strategies for maximizing business growth and profitability. I put together this project to analyze and visualize details being genuinely curious about the findings.

Tools Used: Python, Tableau

Here are some key insights:

  • Kurta Sets, Kurtas, and Western Dresses are the top 3 selling product categories.
  • The top 3 states that experienced maximum sales across all categories are Maharashtra, Karnataka, and Tamilnadu.
  • The top 3 sizes ordered are M, L, and XL.
  • The sales volume of all categories peaks from April to Mid-June.
